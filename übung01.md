# Übung 1
## 1. Welche Behauptungen über Mengen sind wahr/falsch? Begründung?

 * __Ø ⊆ Ø__ 
-> wahr

 * __Ø ∈ Ø__
-> falsch

 * __Ø ∈ { Ø }__ 
 -> wahr, da Definition der leeren Menge (ausreichende Begründung?)

 * __Ø ⊆ { Ø }__
 -> wahr, Leere Menge ist Teilmenge jeder Menge!

 * __Ø ⊆ 2^Ø__
-> wahr

 * __Ø ∈ 2^Ø__
-> wahr
